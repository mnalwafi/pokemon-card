# PokemonCard

This project was generated with Angular CLI version 18.2.3.

## Installation

1. Clone the repository:
   ```gh repo clone mnalwafi/pokemon-card```
   
   ```cd pokemon-card```

3. Install dependencies:
   npm i

## Development Server

To start the development server, run:
   ng s

Then navigate to:
   http://localhost:4200/

The application will automatically reload when you change any of the source files.
